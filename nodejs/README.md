### API/REST
  - **HTTP Routes:** GET, POST, PUT, DELETE;
  - **Advantages:** Multiple front-end, standardized communication,
  - **Route Example:** `http://api.com/company/1/users?page=2`
  - **HTTP codes**
    - **1xx**: Informational
    - **2xx**: Success
      - 200: Success
      - 201: Created
    - **3xx**: Redirection
      - 301: Moved permanently
      - 302: Moved
    - **4xx**: Client Error
      - 400: Bad Request
      - 401: Unauthorized
      - 404: Not Found
    - **5xx**: Server Error
      - 500: Internal Server Error
